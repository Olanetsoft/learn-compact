# Control Flow

Compact provides a small, statically analyzable set of control flow constructs:

- `if` / `if...else` statements
- `for` loops (bounded)
- `return`
- `assert`
- blocks `{ ... }`
- `const` bindings
- conditional expressions `e0 ? e1 : e2`

_Source: [Statements](https://docs.midnight.network/develop/reference/compact/lang-ref#statements)_

- [Conditionals](01-conditionals.md) - `if` statements and ternary expressions
- [Loops](02-loops.md) - bounded `for` loops
