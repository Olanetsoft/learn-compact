# Exercise: Boolean Type

## Goal

Understand and work with Boolean values in Compact.

## Instructions

1. Open `exercise.compact`
2. Create a pure circuit that takes two Boolean parameters
3. Return true if BOTH are true (logical AND)

## Boolean Operations

- `&&` - logical AND
- `||` - logical OR
- `!` - logical NOT

## Verify

```bash
compactc exercise.compact
```

## Hints

<details>
<summary>Hint 1</summary>
Pure circuits use the `pure circuit` keyword.
</details>

<details>
<summary>Hint 2</summary>
Return type is `Boolean` (capital B).
</details>
